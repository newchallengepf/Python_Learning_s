# Python-Portfolio
## 概要
Pythonで売上データを分析するツール。PandasでCSVを処理し、Matplotlibでグラフ化。
## 使い方
1. `data.csv`にデータを入力（例：Month,Sales）。
2. `app.py`を実行。
3. グラフは`output.png`に保存。
